Q5: Suspense and lazy loading:

A: Lazy loading in React means loading components only when they are needed, instead of loading everything upfront. This helps reduce the initial bundle size and improves performance. React provides the React.lazy() function to load components dynamically.

Together, lazy() + Suspense help optimize performance and make React apps load faster.
